# Brennan Brown Repository Analysis (New Set)

This document provides a detailed analysis of a new set of Brennan Brown's GitHub repositories, outlining their purpose, technologies used, and relevant URLs.





### 1. bearminder
- **Repository URL:** https://github.com/brennanbrown/bearminder
- **Demo URL:** (No direct demo URL, but is a macOS menubar app)
- **Description:** A macOS menubar application that tracks daily word counts from the Bear app and posts them to Beeminder goals. It aims to replace the functionality of the now-defunct Draft.com for writers who use Bear for their daily writing and Beeminder for goal tracking. It runs on-demand or hourly, securely stores tokens in Keychain, and focuses on privacy.
- **Technologies:** Swift, macOS AppKit, Beeminder API, Bear App API, Keychain, XcodeGen




### 2. 11ty-Gamification
- **Repository URL:** https://github.com/brennanbrown/11ty-Gamification
- **Demo URL:** https://11ty-gamification.netlify.app
- **Description:** A monorepo containing a blog theme that gamifies writing. It allows users to earn badges for streaks and milestones, visualize activity on a heatmap, and receive nudges via "Badge Ideas." It emphasizes a strong testing culture for confident deployment.
- **Technologies:** Eleventy (11ty), Nunjucks, TypeScript, Node.js, Vitest, Tailwind CSS, npm workspaces




### 3. journalfor.me
- **Repository URL:** https://github.com/brennanbrown/journalfor.me
- **Demo URL:** https://journalforme.netlify.app
- **Description:** An early access, minimalist, secure, Markdown-based journal application with cross-device sync and offline capabilities. It is built as a Progressive Web App (PWA) with zero-knowledge encryption, ensuring privacy by storing only encrypted blobs on the server. It features client-side AES-256 encryption, secure authentication, and a hybrid architecture combining local IndexedDB with encrypted cloud sync.
- **Technologies:** TypeScript, Vite, Tailwind CSS, Netlify Functions, Neon PostgreSQL, JWT, bcrypt, IndexedDB, Service Workers, HTML, CSS, JavaScript




### 4. journalbar
- **Repository URL:** https://github.com/brennanbrown/journalbar
- **Demo URL:** https://journal.kim/
- **Description:** A personal blog for self-quantification and personal improvement, powered by Jekyll. This project serves as a meta-accountability experiment where Brennan Brown provides weekly updates on his goals tracked on Beeminder, aiming to foster consistent reporting and habit adherence.
- **Technologies:** Jekyll, HTML, CSS, Ruby, Shell




### 5. communityandromance
- **Repository URL:** https://github.com/brennanbrown/communityandromance
- **Demo URL:** https://honoursconference2024.com
- **Description:** The official repository for the Mount Royal University English Honours Conference of 2025 (website for the 2024 conference). This website serves as the digital home for the conference, themed around Romance & Community, exploring the intersection of romance and community in literature, relationships, and societal change. It features a responsive design, dynamic schedule display, speaker profiles, and is SEO optimized.
- **Technologies:** Jekyll, HTML, CSS, JavaScript, Ruby, Shell




### 5. cset
- **Repository URL:** https://github.com/brennanbrown/cset
- **Demo URL:** https://cset.netlify.app/
- **Description:** The official website for Calgary's Society for Equity in Technology (CSET). CSET is a volunteer-administered collective offering low-cost technology classes and events to the Calgary community, focusing on creating an inclusive space for marginalized groups and active allies. The website provides information about their mission, events, and resources.
- **Technologies:** Jekyll, HTML, CSS, Ruby, JavaScript, Bootstrap, Font Awesome, Tockify, TypeForm, OpenDyslexic typeface




### 6. moneytree
- **Repository URL:** https://github.com/brennanbrown/moneytree
- **Demo URL:** (No direct demo URL, but is a PWA)
- **Description:** A privacy-first, offline-capable personal finance Progressive Web App (PWA). It supports transactions with accounts and transfers, monthly budgeting, and robust offline caching via a service worker. Data is stored locally in IndexedDB, ensuring privacy with no external APIs.
- **Technologies:** Eleventy (11ty), Tailwind CSS, IndexedDB, JavaScript, Nunjucks, Vitest, Service Workers




### 7. classic-spirits
- **Repository URL:** https://github.com/brennanbrown/classic-spirits
- **Demo URL:** https://classic-spirits.netlify.app
- **Description:** A classic sidebar Jekyll theme for old-school blogging, built with the Bulma framework. It is a fork of Soot Spirits by Abhishek Nagekar, with significant improvements in CSS for better performance, a two-column responsive layout, and integration with Jekyll plugins for RSS, sitemap, accessibility, and SEO.
- **Technologies:** Jekyll, Bulma CSS, HTML, CSS, Ruby, JavaScript




### 8. STA-landing-page
- **Repository URL:** https://github.com/brennanbrown/STA-landing-page
- **Demo URL:** https://brennanbrown.github.io/STA-landing-page/
- **Description:** This repository contains the static landing page for Self-Teach Academy, a centralized dashboard designed to track learning progress across multiple platforms. The project is evolving into a full-stack Next.js application, with the current static page serving as a demo. It aims to empower self-taught students and lifelong learners by automatically ingesting activity from various learning platforms.
- **Technologies:** SCSS, CSS, HTML, JavaScript, TypeScript, Next.js, Prisma, Supabase (for the evolving app)




### 9. kindreadthepoet
- **Repository URL:** https://github.com/brennanbrown/kindreadthepoet
- **Demo URL:** https://kindreadthepoet.com
- **Description:** The professional website of Marcus Madelo, a Filipino-Canadian spoken-word poet and artist based in Calgary, Alberta. The site showcases his work, including a collection of his poems.
- **Technologies:** Jekyll, HTML, CSS, Shell, Ruby




### 10. jekyll-audit
- **Repository URL:** https://github.com/brennanbrown/jekyll-audit
- **Demo URL:** https://brennanbrown.github.io/jekyll-audit/
- **Description:** An npm package and CLI tool designed to audit Jekyll sites for performance, accessibility, SEO, broken links, and HTML validation. It builds and serves the Jekyll site locally, then runs various audits using tools like Lighthouse, Pa11y, Linkinator, and html-validator, providing consolidated reports.
- **Technologies:** TypeScript, JavaScript, Node.js, npm, Jekyll, Lighthouse, Pa11y, Linkinator, html-validator
